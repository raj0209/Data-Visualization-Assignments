HOMELESS POPULATION IN USA

Name: Raj Buddhadev       Course: Data Visualization (CSE 578)       Date: 15th October 2018

The choropleth map shows the number of homeless people in each state of USA in 2016. While, the line chart shows the percentage increase/decrease in the number of homeless people from 2007-2016 in the country.

When the user hovers on any state, he/she can see the number of homeless people in that state in 2016. On clicking a particular state, a line chart showing the percentage increase/decrease in number of homeless people from 2007-2016 in that state will be displayed.

For example, if the user hovers the mouse on the state 'Arizona', the pop up box shows the number of homeless people in Arizona in 2016. On clicking the state of Arizona, the line chart showing the percentage increase/decrease in number of homeless people in Arizona from 2007-2016 will be displayed.

Extra Credit : For extra credit on this assignment, I have also implemented a choropleth map that shows the per capita value of homeless people(percentage of homeless people) in each state in 2016. This percentage is obtained by dividing the number of homeless people in a state in 2016 by the total population of that state in 2016. Hover and on-click interactions are also applied on the per-capita choropleth map. Two maps can be toggled through the drop down menu present on top left of the map.
